# Pineapple Schedule - A simple schedule web for CCU students

## Local development

### Prerequisites

- [Node.js](https://nodejs.org/en/) (>= 10.16.0)
- [npm](https://www.npmjs.com/) (>= 6.9.0)

### Installation

```bash
npm install
```

### Run

```bash
使用 run.ps1
```
或者確保vue已經打包成靜態物建以及tailwind動態更新後
```bash
cd ./server
npm start
```

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```

### Tailwind CSS Hot Reload

```bash
npm run hotfix
```
